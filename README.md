# Shopify Backend Challenge



## Project Setup

1. Clone this repository
2. Run `npm start`
3. Visit `http://localhost:3000/`

## Dependencies

- Node 5.10.x or above
- NPM 3.8.x or above
- Express 4.15.x or above
- Nodemon 1.11.x or above
